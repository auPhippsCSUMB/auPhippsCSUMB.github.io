A (Brief) Description of CSS


CSS (Cascading Style Sheets) is style sheet language that is designed to style another document written in a markup language. CSS is most commonly used to style HTML or XML. CSS works by specifying an element from the document being styled and applying values to said elements. Before CSS, webpage authors would often have to describe the style of each individual html element, oftentimes repeatedly. Since its initial release in 1996, CSS has been used by millions of webpage programmers to design their work. Below are a few examples of CSS in action.
