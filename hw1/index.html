A Short Overview of HTML


HTML is the essential core of what makes a website function. It is the language that websites are built upon. HTML means Hyper Text Markup Language and, as the name implies, it is a type of markup language that uses hypertext links to provide navigation to other webpages, the basis of the internet. HTML largely provides the structure of the webpage, dictating elements with angled bracket tags <h1> <----- such as this. More in depth design such as precise positioning, sizing, and color is done with CSS assistance. Furthermore, more advanced functions are often programmed with JavaSript and inserted seamlessly into HTML websites, often for more user interaction.
